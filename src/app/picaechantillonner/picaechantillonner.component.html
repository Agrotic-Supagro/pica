Pour placer les points, il faut cliquez sur l'icone puis cliquer sur la carte là où vous voulez échantillonner. Il faut réaliser cette opératon pour chaque point.
Pour déplacer le point, cliquez sur le point puis garder le clic enfoncé en déplaçant la souris. Relachez le bouton pour placer le point.
Pour supprimer un point, cliquez sur le point puis cliquez sur le bouton "Suppr." dans la popup qui s'est ouverte.
